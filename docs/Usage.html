1 Usage

ip-address is an Emacs package for looking up your IP address.

1.1 ip-address

M-x ip-address will do an IP address lookup.

This is accomplished by getting the contents of https://html.duckduckgo.com/html?q=ip%20address, extracting the IP address information and displaying it in the minibuffer.
